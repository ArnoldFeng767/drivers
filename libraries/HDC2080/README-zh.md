# HDC2080

**课程参考：**

```python
from hdc2080 import Hdc2080
``````


**实例参数：**

| 名称     | 必填项 | 类型 | 描述            |
| -------- | ---- | ---- | ----------------------- |
| i2c_obj  | 是  | 整型  | I2C 对象                |
| dev_addr | 否  | 整型  | I2C 从设备地址，默认值为 0x40 |

```python
i2c_dev = I2C(I2C.I2C1, I2C.STANDARD_MODE)
hdc = Hdc2080(i2c_dev)
```


**接口功能：**

l **read()**

读取寄存器中的值，并将其转换为湿度和温度。

参数：


否。


返回值：

| 名称                   | 类型  | 描述      |
| ---------------------- | ----- | ---------- |
| （湿度，温度） | 元组 | 湿度、温度 |

l **重置（函数）**

重置 hdc1080 。

参数：


否。


返回值：

否。
